# Python Project Management CLI Tool

A Python-based command-line application for managing users, projects, and tasks with local JSON persistence.

## Features

- Add and list users
- Add and list projects for specific users
- Add and list tasks for projects
- Mark tasks as complete
- Search for a project by title
- Persist data with JSON file storage
- Beautiful terminal UI using Rich
- Unit tests using pytest

## Tech Used

- Python 3.10+
- argparse
- json
- rich
- python-dateutil
- pytest
